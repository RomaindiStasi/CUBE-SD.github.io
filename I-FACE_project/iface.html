Project Overview

The iFACE project is part of the broader CUBE-SD program and builds upon the PhD research of Romain di Stasi. This work focused on how to reliably measure surprise in infants within social contexts, combining both physiological measures and behavioral data. Behavioral responses were assessed using a multimethod approach, including manual coding (via The Observer XT ©) and automated procedures designed to quantify facial expressions of surprise. The corresponding article, currently under revision in Acta Psychologica, is available as a preprint on SSRN and PsyArXiv (see links below).

In this research, Dr. Romain di Stasi and Dr. Fabien Cerrotti developed three complementary algorithms, implemented within the BabyFace software ©, to detect and quantify facial expressions of surprise. These algorithms are grounded in well-established Facial Action Coding System (FACS) markers characterizing surprise expressions.

Specifically, the models target three key facial components: (1) eyebrow raising (Action Units AU1 and AU2), (2) eye widening (AU5), and (3) mouth opening (AU26 and AU27). These features were grouped into three functional facial regions: the upper face (eyebrows and forehead: AU1 & AU2), the eye region (AU5), and the lower face (mouth opening: AU26 and AU27). This regional approach allows for a more robust and interpretable quantification of surprise-related facial dynamics in infants.

The iFACE project aims to provide an open-access implementation of these algorithms. The computational framework is primarily based on Python, relying on several key libraries for computer vision and machine learning (e.g., OpenCV, deep learning frameworks, and facial landmark detection tools), with the objective of ensuring reproducibility, scalability, and accessibility for the research community.

Methods

The iFACE project relies on a multidimensional methodological approach combining behavioral coding and automated facial analysis to study infant emotional expressions, with a particular focus on surprise.

The training dataset is based on the work of Webb et al. (2017), who compiled a set of infant facial expressions categorized into six prototypical emotions (e.g., surprise, joy, sadness). While the original dataset included 154 unique images, the final dataset used in our analyses consisted of 687 image files, including multiple variants (e.g., different crops, resolutions, or augmentations).

For model development, the dataset was split into a training set (80%, n = 552 images) and a validation set (20%, n = 138 images). This split allowed us to evaluate model performance while limiting overfitting.

To assess model robustness, we implemented an overfitting evaluation strategy based on performance metrics. A high risk of overfitting was identified when the support was low (< 10) and F1-scores exceeded 0.90, while intermediate thresholds defined moderate risk. Training configurations were iteratively adjusted to encourage the learning of invariant features and improve generalization.

How to Contribute

We welcome contributions from the research community to improve and extend the iFACE project. Whether you're interested in improving the algorithms, adding new features, or reporting issues, your input is valuable.

1 Report Issues & Bugs

If you encounter any bugs or issues while using the iFACE algorithms, please report them via the GitHub Issues page. Include detailed information about the error, your environment, and steps to reproduce.

📝 Report an Issue

2 Contribute Code

We accept pull requests for bug fixes, new features, and documentation improvements. Please ensure your code follows the project's coding standards and includes appropriate tests.

  • Fork the repository
  • Create a feature branch
  • Submit a pull request

3 Share Your Data

If you have infant facial expression datasets that could help improve model performance, we'd love to hear from you. Please contact us to discuss data sharing agreements and ethical considerations.

4 Cite Our Work

If you use the iFACE algorithms in your research, please cite our preprint:

di Stasi, R., Cerrotti, F., Esseily, R., & Rat-Fischer, L. (2026). Unveiling Infant Surprise in a Social Context: A Multidimensional Analysis of Physiological, Behavioural, and Facial Expression Markers. Preprint on SSRN. https://doi.org/10.31234/osf.io/kshw3_v1

5 Join the Discussion

Have questions or ideas? Join our community discussions on GitHub or reach out directly via email.
